Bucindolol is a b-blocker / sympatholytic agent whose efficacy is modulated by polymorphisms in the primary target ( b ( 1 ) adrenergic receptor [ AR ] Arg389 Gly on cardiac myocytes ) and a secondary target modifier ( a ( 2C ) AR Ins [ wild-type ( Wt ) ] 322-325 deletion [ Del ] on cardiac adrenergic neurons ) .
